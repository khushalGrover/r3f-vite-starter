# r3f-vite-starter
A boilerplate to build R3F projects

## Live demo
https://khushalgrover.github.io/r3f-vite-starter/

## To run locally
```
yarn
yarn dev
```


## To Deploy 
npm i gh-pages

```
yarn build
yarn deploy
```
